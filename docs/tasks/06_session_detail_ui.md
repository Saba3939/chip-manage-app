# Task 06: セッション詳細UI

## 目的
セッション画面のUIを構築し、参加者リスト、残高表示、QRコード表示などを実装する。

## 実装内容

### 1. セッション詳細画面（`app/(main)/session/[id]/page.tsx`）

**ステータスによる表示切り替え**

#### 待機中（waiting）
- セッション情報表示
  - セッション名
  - 初期チップ数
  - 現在の参加人数 / 最大人数
- QRコード表示
  - 参加用URL: `/session/join/{sessionId}`
  - 「QRコードを表示」ボタン → ダイアログで大きく表示
- 参加者リスト
  - アバター
  - 表示名
  - 参加時刻
- ホストのみ表示
  - 「ゲームを開始」ボタン（2人以上で有効化）
  - 「セッションをキャンセル」ボタン

#### 進行中（active）
- 自分の残高（大きく表示）
- 参加者リスト
  - アバター
  - 表示名
  - 現在のチップ数
  - 「送る」ボタン → チップ送信ダイアログ
- チップ総数表示（検証用）
  - 初期チップ × 参加者数 = 合計チップ
- ホストのみ表示
  - 「セッションを終了」ボタン

#### 完了（completed）
- 清算画面へ自動遷移

### 2. コンポーネント設計

**`components/session/SessionHeader.tsx`**
- セッション名
- ステータスバッジ
- 戻るボタン

**`components/session/ParticipantList.tsx`**
- 参加者カードのリスト表示
- ステータスに応じた表示切り替え

**`components/session/ParticipantCard.tsx`**
- 個別の参加者情報カード
- 残高表示（activeステータス時）
- 送るボタン（activeステータス時）

**`components/qr/QRCodeDisplay.tsx`**
- QRコード生成（`qrcode.react`使用）
- ダイアログ表示
- URLコピー機能

**`components/session/BalanceDisplay.tsx`**
- 自分の残高を大きく表示
- アニメーション効果

### 3. モックデータの拡張
- 参加者の残高データ
- トランザクション履歴

## 使用コンポーネント
- Card
- Badge（ステータス表示）
- Dialog（QRコード表示用）
- Button
- Avatar
- Separator
- Progress（参加人数ゲージ）

## 外部ライブラリ
- `qrcode.react`: QRコード生成
