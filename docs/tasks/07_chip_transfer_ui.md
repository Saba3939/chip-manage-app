# Task 07: チップ送受信UI

## 目的
チップを送受信するためのUIを構築し、モックで送信フローを確認する。

## 実装内容

### 1. チップ送信ダイアログ

**`components/chip/ChipTransferDialog.tsx`**

**表示項目**
- ダイアログタイトル: 「チップを送る」
- 受取人情報
  - アバター
  - 表示名
  - 現在のチップ数
- 送信者情報（自分）
  - 現在のチップ数
  - 送信後の残高（プレビュー）
- 送信額入力
  - 数値入力フィールド
  - クイック選択ボタン（100, 500, 1000, All-in）
- 確認メッセージ
  - 「{受取人}に{金額}チップを送りますか？」
- アクションボタン
  - 送信ボタン（プライマリ）
  - キャンセルボタン

**バリデーション**
- 送信額 > 0
- 送信額 <= 自分の残高
- 整数のみ

**モック動作**
- 送信ボタンクリック → ローカルステートで残高を更新
- 成功トースト表示
- ダイアログを閉じる

### 2. クイック送信ボタン

**`components/chip/QuickAmountButtons.tsx`**
- 定額ボタン（100, 500, 1000）
- All-inボタン（全額送信）
- ボタンクリックで入力フィールドに金額がセット

### 3. トランザクション履歴表示

**`components/chip/TransactionHistory.tsx`**
- 最近のトランザクション表示（5-10件）
- 表示項目
  - 時刻
  - 送信者 → 受取人
  - 金額
  - アイコン（送信/受信を色分け）

### 4. 状態管理（モック）
- ローカルステート（useState）で残高を管理
- トランザクション配列を管理
- 送信時に両者の残高を更新

### 5. アニメーション・フィードバック
- 残高変更時のアニメーション
- 送信成功時のトースト通知
- ローディング状態（ボタン無効化）

## 使用コンポーネント
- Dialog, DialogContent, DialogHeader, DialogTitle
- Input
- Button
- Avatar
- Badge
- Toast（shadcn/ui）
- Card（履歴表示）
